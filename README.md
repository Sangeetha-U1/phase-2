# NovaTech - Responsive Company Landing Page

## Project Overview

NovaTech is a modern and responsive company landing page built using HTML5 and CSS3.

This project focuses on:

* Responsive Web Design
* Flexbox Layouts
* CSS Grid
* Modern UI/UX Design
* Mobile-Friendly Navigation
* Gradient Backgrounds
* Hover Effects
* Responsive Images

The landing page is designed for startups, businesses, and creative agencies.
